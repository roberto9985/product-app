# Product Management CRUD Application

This is a simple **Spring Boot** application to manage products
It supports **CRUD operations**: create, read, update and delete products.

The application uses **method-level authentication**, **logging**, **unit testing**, and **global exception handling**. It also uses an **H2 in-memory database** for simplicity.

---

## Features

- Create a product
- Read all products or a product by ID
- Update a product
- Delete a product 
- **Basic authentication** on service methods 
- **Logging** with SLF4J
- **Global exception handling**
- **Unit testing** with JUnit 5 and Mockito

## Technologies

- Java 21
- Spring Boot 3.x
- H2 database (in-memory)
- Spring Security
- SLF4J
- JUnit
- Mockito
- SLF4J
- Hibernate/JPA
